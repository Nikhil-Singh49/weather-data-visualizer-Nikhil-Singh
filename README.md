Weather Data Analysis

This project processes raw weather data and produces cleaned datasets, statistical summaries, and visual visualizations. It is designed to work with CSV files that may use different column names for date, temperature, humidity, and rainfall.

Features

Automatic detection and normalization of column names

Cleaning and standardization of weather data

Synthetic rainfall generation if rainfall is missing

NumPy-based temperature statistics

Monthly and seasonal aggregation

Visualizations:

Daily temperature trend
Monthly rainfall totals

Humidity vs. temperature scatterplot

Combined temperature + rainfall view
